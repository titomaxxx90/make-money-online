Будем SaaS на Github Pages

Максим Прохоров 10/03/2026

Шаг 1: Выяснение требований

Чтобы написать идеальный код, нам нужно:

  1. Стек технологий:  комфортнее работать с современным фреймворком вроде React (он идеально подходит для таких форм и переключений)  В своем решении ниже я буду использовать React, так как он лучше всего подходит для поставленных задач.

  2. База данных:  аккаунт в Supabase 

  3. Специфика стран: Для разных стран (РФ, РК, РБ, КР) формы сильно отличаются. Ищем образцы или списки обязательных полей для каждой страны. (Например, в РФ нужен ИНН/КПП, а в РК — БИН/ИИН и КБЕ).

Шаг 2: Описание решения

Мы создадим Single Page Application (SPA) на базе React.

  • Интерфейс (Frontend): React + Tailwind CSS (для быстрого и красивого современного дизайна, который сам адаптируется под телефоны и ПК).

  • База данных (Backend): Supabase. Он возьмет на себя регистрацию пользователей (Auth) и хранение истории выставленных счетов в базе данных PostgreSQL.

  • Экспорт: Мы будем использовать библиотеки jspdf (для создания PDF) и xlsx (для Excel).

  • Хостинг: GitHub Pages отлично работает со статичными сборками React.

Мои идеи, что можно добавить (дополнительный функционал):

  • Автозаполнение: При вводе БИН/ИНН автоматически подтягивать название компании из открытых API.

  • QR-код для оплаты: Автоматическая генерация QR-кода на счете для быстрой оплаты через банковские приложения.

  • Дашборд: Простой график на главной странице, показывающий сумму выставленных счетов за месяц.

  • Отправка на почту: Кнопка "Отправить клиенту", которая сразу шлет PDF на email.

Шаг 3: Структура файлов

Для начала работы тебе понадобится установить Node.js на компьютер. Мы создадим проект с помощью сборщика Vite (это современный и быстрый инструмент).

Структура нашего проекта будет выглядеть так:

Шаг 4: Базовый код приложения (MVP)

Мы напишем фундамент — основной код, который демонстрирует подключение базы, форму с переключением стран и логику работы.

1. Подключение к Supabase (src/supabaseClient.js)

Этот файл связывает твое приложение с базой данных. Тебе нужно будет вставить сюда свои ключи из панели Supabase.

JavaScript
 
import { createClient } from '@supabase/supabase-js';

// Замени эти данные на свои из настроек проекта Supabase (Project Settings -> API)
const supabaseUrl = 'ТВОЙ_SUPABASE_URL';
const supabaseKey = 'ТВОЙ_SUPABASE_ANON_KEY';

export const supabase = createClient(supabaseUrl, supabaseKey);

2. Главная форма создания счета (src/components/InvoiceForm.jsx)

Здесь мы реализуем переключение между странами и сбор данных.

JavaScript
 
import { useState } from 'react';
import { supabase } from '../supabaseClient';
// Функции экспорта, которые мы создадим позже
import { exportToPDF, exportToExcel } from '../utils/exportDocs'; 

export default function InvoiceForm({ user }) {
  // Состояние для выбранной страны
  const [country, setCountry] = useState('РК');
  
  // Состояние для данных формы
  const [formData, setFormData] = useState({
    clientName: '',
    amount: '',
    taxId: '', // ИНН или БИН
    description: 'Оплата за услуги веб-разработки'
  });

  // Функция для обновления данных при вводе
  const handleChange = (e) => {
    setFormData({ ...formData, [e.target.name]: e.target.value });
  };

  // Функция сохранения в историю (в Supabase)
  const handleSaveToHistory = async () => {
    const { data, error } = await supabase
      .from('invoices')
      .insert([
        { 
          user_id: user.id, 
          country: country, 
          client_name: formData.clientName, 
          amount: formData.amount,
          details: formData
        }
      ]);
      
    if (error) alert('Ошибка сохранения: ' + error.message);
    else alert('Счет успешно сохранен в историю!');
  };

  return (
    <div className="max-w-2xl mx-auto p-6 bg-white shadow-lg rounded-lg mt-10">
      <h2 className="text-2xl font-bold mb-6 text-gray-800">Создание счета / АВР</h2>
      
      {/* Кнопки переключения стран */}
      <div className="flex space-x-2 mb-6">
        {['РК', 'РФ', 'РБ', 'КР'].map(c => (
          <button 
            key={c}
            onClick={() => setCountry(c)}
            className={`px-4 py-2 rounded ${country === c ? 'bg-blue-600 text-white' : 'bg-gray-200 text-gray-700'}`}
          >
            {c}
          </button>
        ))}
      </div>

      {/* Форма */}
      <div className="space-y-4">
        <div>
          <label className="block text-sm font-medium text-gray-700">Название компании клиента</label>
          <input type="text" name="clientName" onChange={handleChange} className="mt-1 block w-full border border-gray-300 rounded-md p-2" />
        </div>
        
        <div>
          <label className="block text-sm font-medium text-gray-700">
            {country === 'РК' ? 'БИН/ИИН' : 'ИНН'} клиента
          </label>
          <input type="text" name="taxId" onChange={handleChange} className="mt-1 block w-full border border-gray-300 rounded-md p-2" />
        </div>

        <div>
          <label className="block text-sm font-medium text-gray-700">Сумма</label>
          <input type="number" name="amount" onChange={handleChange} className="mt-1 block w-full border border-gray-300 rounded-md p-2" />
        </div>
      </div>

      {/* Кнопки действий */}
      <div className="mt-8 flex flex-wrap gap-4">
        <button onClick={() => exportToPDF(formData, country)} className="bg-red-500 text-white px-4 py-2 rounded hover:bg-red-600">
          Скачать PDF
        </button>
        <button onClick={() => exportToExcel(formData, country)} className="bg-green-500 text-white px-4 py-2 rounded hover:bg-green-600">
          Скачать Excel
        </button>
        <button onClick={handleSaveToHistory} className="bg-indigo-500 text-white px-4 py-2 rounded hover:bg-indigo-600">
          Сохранить в историю
        </button>
      </div>
    </div>
  );
}

Шаг 5: Инструкция по сборке и запуску (Первые шаги)

Чтобы этот код ожил на твоем компьютере, выполни следующие простые шаги в терминале (командной строке):

  1. Создай проект: Открой терминал и введи: npm create vite@latest invoice-app -- --template react Затем перейди в папку проекта: cd invoice-app

  2. Установи необходимые библиотеки: Нам понадобятся инструменты для работы с базой данных, дизайном и экспортом. npm install @supabase/supabase-js jspdf xlsx html2canvas npm install -D tailwindcss postcss autoprefixer npx tailwindcss init -p

  3. Запусти проект локально: npm run dev Открой в браузере ссылку http://localhost:5173 — там появится твое приложение!

  4. Подготовка базы данных: В панели Supabase создай новую таблицу с названием invoices. Добавь в неё колонки: id (уже есть), user_id (uuid), country (text), client_name (text), amount (numeric), details (jsonb).
